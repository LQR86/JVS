# 目标

挣到100万美金

# 核心策略

## 方法论

- 赛道筛选
  - 利用信息差
  - 能解决某种市场需求或痛点
  - 能形成飞轮效应
- 100万美金 = X笔成交 * Y单笔成交利润
  - 专注客户开发（寻找X笔成交）
    - 专业的人做专业的事
    - 避免全链路带来的冷启动成本和风险
    - 专注积累客户资源
    - 利用客户资源带来的飞轮效应

## 0-1阶段

调研供应商（及赛道），做代理，开发客户，积累客户资源

## 1-100阶段

- 通过积累客户资源加强供应链话语权，降低成本，逐步形成飞轮效应，逐步可以对接更多供应商，进一步弱化供应商的话语权
- 在客户资源足够多的情况下，考虑自建后端供应链（如果还没挣到100万美金的话）

# 执行

## 方法论

- 目标导向
- KPI导向
- 问题导向


## SOP

- 根据目标、核心策略选择赛道，寻找供应商
- 确定供应商的情况下，制定KPI
- 根据KPI制定每天的TODO
- 执行TODO，已完成的将其关闭
- 统计分析KPI完成情况
- 在必要的情况下调整KPI体系/策略/赛道
